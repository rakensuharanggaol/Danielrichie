HTML (Hypertext Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML menggambarkan struktur halaman web secara semantik dan terdiri dari serangkaian elemen yang memberi tahu browser cara menampilkan konten.

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk menggambarkan presentasi dokumen HTML atau XML. CSS menjelaskan bagaimana elemen harus ditampilkan di layar, di kertas, atau di media lain. Dengan CSS, Anda dapat mengontrol tata letak, warna, dan font elemen HTML.
